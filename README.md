## Customization
Edit the CSS variables in `:root` to adjust colors:

```css
--neon-cyan: #00ffff;        /* Primary accent */
--moss-green: #50c878;       /* Secondary accent */
--midnight-bg: #0a0e27;      /* Background */
```
